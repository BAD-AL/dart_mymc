# test/test_files — Reference Files

All files in this directory are used by `test/dart_mymc_test.dart`.
Binary reference files were generated from the Python 2.7 `mymc` implementation
(`mymc-pysrc-2.7/mymc.py`) using `~/Programs/python2/bin/python2`.

---

## Memory Card Images

| File | Description | How generated |
|---|---|---|
| `Mcd001.ps2` | Primary test card; contains 5 real PS2 saves | Real PCSX2 memory card image |
| `Mcd_test.ps2` | Scratch copy of Mcd001 used by Phase 3 write tests | Copy of Mcd001.ps2 |
| `blank.ps2` | A freshly formatted 8 MB card image | `mymc.py blank.ps2 format` |
| `blank.7z` | Compressed `blank.ps2` (not currently used) | `7z a blank.7z blank.ps2` |

## Golden Output Files

Generated by running the Python reference implementation and capturing stdout.
Used for exact character-for-character CLI output parity checks.

| File | Command used to generate | Test that uses it |
|---|---|---|
| `golden_df.txt` | `mymc.py test/test_files/Mcd001.ps2 df` | `df output matches Python golden file` |
| `golden_ls.txt` | `mymc.py test/test_files/Mcd001.ps2 ls` | `ls output matches Python golden file` |
| `golden_dir.txt` | `mymc.py test/test_files/Mcd001.ps2 dir` | `dir output matches Python golden file` |

To regenerate all three golden files:
```sh
PY2=~/Programs/python2/bin/python2
MYMC=mymc-pysrc-2.7/mymc.py
CARD=test/test_files/Mcd001.ps2
$PY2 $MYMC $CARD df  > test/test_files/golden_df.txt
$PY2 $MYMC $CARD ls  > test/test_files/golden_ls.txt
$PY2 $MYMC $CARD dir > test/test_files/golden_dir.txt
```

> **Note:** Golden files must be regenerated if `Mcd001.ps2` is replaced.
> Do not regenerate from the Dart implementation — always use Python as the source of truth.

## Regenerating from Scratch

If you need to recreate all Python-derived reference files from `Mcd001.ps2`:

```sh
PY2=~/Programs/python2/bin/python2
MYMC=mymc-pysrc-2.7/mymc.py
CARD=test/test_files/Mcd001.ps2

# Golden output
$PY2 $MYMC $CARD df  > test/test_files/golden_df.txt
$PY2 $MYMC $CARD ls  > test/test_files/golden_ls.txt
$PY2 $MYMC $CARD dir > test/test_files/golden_dir.txt

# PSU oracle
$PY2 $MYMC $CARD export -o test/test_files/NFL2K16.psu /BASLUS-20919NFL2K16

# MAX oracle
$PY2 $MYMC $CARD export -t max -o test/test_files/NFL2K16.max /BASLUS-20919NFL2K16
```
